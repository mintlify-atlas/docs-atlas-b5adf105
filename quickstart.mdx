---
title: "Quickstart"
description: "Get Tooltify running in your React app in under 5 minutes."
---

<Steps>
  <Step title="Install the packages">
    Install `@tooltify/core` and the bundler integration for your project.

    <CodeGroup>
      ```bash npm
      npm install @tooltify/core @tooltify/integration-rspack
      ```
      ```bash yarn
      yarn add @tooltify/core @tooltify/integration-rspack
      ```
      ```bash pnpm
      pnpm add @tooltify/core @tooltify/integration-rspack
      ```
    </CodeGroup>

    <Note>
      `@tooltify/integration-rspack` has a peer dependency on `@rspack/core ^1.7.8`. Make sure it is already installed in your project.
    </Note>
  </Step>

  <Step title="Create tooltify.config.json">
    Create a `tooltify.config.json` file in your project root (next to `package.json`). This file is required — the server and CLI both read it from `process.cwd()`.

    ```json tooltify.config.json
    {
      "packagesDir": "./src",
      "port": 4100,
      "auth": {
        "secret": "change-me-in-production",
        "salt": "change-me-in-production"
      }
    }
    ```

    <Warning>
      Use strong random values for `secret` and `salt` in any shared environment. Do not commit these values to version control.
    </Warning>

    | Field | Description |
    |---|---|
    | `packagesDir` | Path to your source directory (relative to project root). The editor API only serves files inside this directory. |
    | `port` | Port the Tooltify server listens on. Defaults to `4100` if omitted. |
    | `auth.secret` | Secret used for JWT signing. |
    | `auth.salt` | Salt used for password hashing. |
  </Step>

  <Step title="Add the Rspack plugin">
    Add `rspackTooltify()` to your `rspack.config.ts`. The plugin starts the Tooltify server, replaces the React JSX dev runtime to enable source tracking, and injects the client bundle into your HTML.

    ```typescript rspack.config.ts
    import { HtmlRspackPlugin } from "@rspack/core";
    import { rspackTooltify } from "@tooltify/integration-rspack";
    import { createRequire } from "module";

    const require = createRequire(import.meta.url);

    export default {
      entry: "./src/index.tsx",

      output: {
        publicPath: "/"
      },

      module: {
        rules: [
          {
            test: /\.tsx?$/,
            loader: "builtin:swc-loader",
            options: {
              jsc: {
                parser: { syntax: "typescript", tsx: true },
                transform: {
                  react: { runtime: "automatic" }
                }
              }
            }
          }
        ]
      },

      resolve: {
        extensions: [".tsx", ".ts", ".js"],
        alias: {
          "react": require.resolve("react")
        }
      },

      plugins: [
        new HtmlRspackPlugin({ template: "./index.html" }),
        rspackTooltify({
          react: {
            // Only inject source info into styled-components elements (optional)
            shouldInjectSource: (type) => {
              return typeof type === "object" && typeof type?.target === "string"
            },
          }
        })
      ],

      devServer: {
        port: 5173,
        historyApiFallback: true,
      }
    };
    ```

    <Tip>
      The `shouldInjectSource` callback is optional. When omitted, Tooltify injects source attributes on all DOM elements (any component whose `type` is a string). Pass a custom predicate to restrict injection — for example, to styled-components only as shown above.
    </Tip>
  </Step>

  <Step title="Start the agent daemon">
    Run `tooltify` from your project root to launch the interactive agent manager. The agent connects to the Tooltify server over WebSockets and handles IDE-open commands from the browser.

    ```bash
    npx tooltify
    ```

    You will see an interactive menu:

    ```
    ◆  Tooltify Agent Manager
    │
    ◆  What do you want to do?
    │  ● Start agent (alice)
    │    Stop agent (alice)
    │    Reset password (alice)
    │    View status
    │    View logs
    └
    ```

    Select **Start agent (alice)**. On first run, you will be prompted to set a password. Then choose your IDE (VS Code or Antigravity) and whether you are connecting remotely.

    Once started, the agent runs as a background daemon and reconnects automatically if the server restarts.
  </Step>

  <Step title="Open your app in the browser">
    Start your Rspack dev server (adjust the command to match your project setup):

    ```bash
    npx rspack serve
    ```

    Open `http://localhost:5173` (or your configured dev server port). The Tooltify devtools panel is now embedded in your app. Click any element to inspect its source location and open it directly in your IDE.
  </Step>
</Steps>
